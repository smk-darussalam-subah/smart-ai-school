# Aset Foto Landing Page

Ganti file-file berikut dengan foto asli dari IG @smkdarussalamsubah / galeri sekolah.
Ukuran rekomendasi: AVIF/WebP, ≤ 200 KB per file.

| File              | Konten                                     | Ukuran ideal |
|-------------------|--------------------------------------------|--------------|
| hero-main.jpg     | Suasana belajar / praktik (foto utama)     | 800×600      |
| hero-santri.jpg   | Kegiatan santri / ibadah                   | 400×400      |
| hero-lab.jpg      | Praktik bengkel / lab komputer             | 400×400      |

Sumber foto valid:
- IG @smkdarussalamsubah
- Google Maps / GMaps (foto tempat)
- Koleksi internal sekolah

JANGAN gunakan foto stok / unsplash.
