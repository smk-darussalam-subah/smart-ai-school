Opsi B — Scroll Tabs + Affordance

3 refinemen gold-standard yang membedakan dari horizontal-scroll biasa:

  1. Edge fade gradient di tepi kanan — sinyal visual "masih ada tab →" (fix discoverability, pelanggaran Nielsen pada versi sebelumnya).
  2. Auto-scroll active — tab aktif otomatis center saat ganti screen (lihat: Penilaian aktif & terlihat meski di tengah list).
  3. Sticky nav — tetap visible saat scroll konten (konten demo bergaris di bawah).
  4. Filter pill kompak — bukan 5 chip terpisah. 1 pill, tap = expand full filter sheet.

Bandingkan: device kiri = SEBELUMNYA (bar bertumpuk). Device kanan = OPSI B (enhanced).
