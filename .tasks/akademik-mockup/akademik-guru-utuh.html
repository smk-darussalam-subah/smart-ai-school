Struktur Nilai & Alur Data
Komposisi Nilai Akhir (NA): UH×20% + Praktik×25% + Sikap×15% + UTS×20% + UAS×20%. UH = rata² formatif harian TP. Praktik = tugas/proyek/observasi. Sikap = observasi langsung.
Formatif harian (TP) → rata² → UH di Gradebook
Sumatif per CP → ukur ketercapaian CP
UTS = sumatif beberapa CP tahap 1 · UAS = sumatif semua CP tahap 1+2
NA ≥ KKTP → CP terkait bertambah progres → Rapor
Semua agregasi → Rekap Audit KS/Wakakur

MODUL AJAR

Kurikulum Merdeka — SMK Darussalam Subah
Mata Pelajaran: Pemrograman Web
Kelas: XI TJKT 1
Fase: Fase F (Kelas XI-XII)
Semester: Ganjil
Alokasi JP: 6 JP (45 menit/JP)
KKTP: 75
Pengembang: Budi Hartono, S.Kom
Judul: Flexbox & Desain Responsif
Capaian Pembelajaran (CP)
Pada akhir fase F, peserta didik mampu merancang antarmuka web yang responsif menggunakan CSS flexbox, media query, dan teknik layout modern.
Tujuan Pembelajaran (TP)
  • TP 1: Menjelaskan konsep flexbox dan properti utamanya
  • TP 2: Menerapkan justify-content dan align-items untuk menyusun layout
  • TP 3: Membuat halaman web responsif menggunakan media query
Alur Tujuan Pembelajaran (ATP)
  • TP 1 → Pertemuan 1-2: Menyebutkan minimal 3 properti flexbox
  • TP 2 → Pertemuan 3-4: Membuat layout navigasi horizontal
  • TP 3 → Pertemuan 5-6: Membuat halaman responsif
Profil Pelajar Pancasila
Bernalar Kritis, Mandiri. Melalui praktik mandiri merancang layout responsif, peserta didik mengembangkan kemandirian dan bernalar kritis.
Kegiatan Pembelajaran
Pendahuluan (15 menit): Apersepsi, motivasi, tujuan pembelajaran.
Inti (60 menit): Demonstrasi flexbox, praktik, peer review, media query.
Penutup (15 menit): Kesimpulan, refleksi, tindak lanjut.
Asesmen
Diagnostik: 2 soal PG (flexbox, CSS)
Formatif: 2+ soal PG + praktikum layout responsif
Sumatif: 20-30 soal PG + Proyek akhir (Web Profil Sekolah Responsif)
Pengayaan & Remedial
Pengayaan: CSS Grid, lab layout tingkat lanjut.
Remedial: Tutor sebaya, materi ringkas, latihan ulang. Nilai maks = KKTP.
Lampiran
3 file: Handout PDF, Slide PPTX, Lembar Observasi XLSX
